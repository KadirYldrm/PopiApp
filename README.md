# PopiApp with MVVM

#### This app offers you popular movies, serials and Artists. It also keeps you up to date with the update option.

## TECH

- Kotlin
- Room
- Dagger
- Retrofit
- Glide
- Coroutines
- View Binding
- TMDB API

### This app is written using Dependency Injection and Clean Architecture.
